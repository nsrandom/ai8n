Create a SQLite db for the following schema:

Workflows:
	id (primary key)
	name (text)
	active (boolean)
	created_at (timestamp)
	updated_at (timestamp)


Node:
	id (primary key)
	workflow_id (foreign key to Workflows)
	name (text)
	type (text)  -- can be "Trigger", "Command", "Constant", "LLM", "Conditional", "Manual"
	parameters (json / text)
	position (json / text)
	created_at (timestamp)
	updated_at (timestamp)

Connections:
	id (primary key)
	from_node_id (foreign key to Workflows)
	to_node_id (foreign key to Workflows)
	created_at (timestamp)
	updated_at (timestamp)


Executions:
	id (primary key)
	workflow_id (foreign key to Workflows)
	node_id (foreign key to Nodes)
	run_index (int)
	status (text)
	started_at (timestamp)
	ended_at (timestamp)
	input (json / text)
	output (json / text)
	error (json / text)
